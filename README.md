# residential alarm system, zigbee sensors, tcp control, video monitoring and recording

Python-based residential alarm system

## Features
- No DNS required

## Project Structure
- `server.py` - runs on the server machine
- `client.py` - runs on the client
- `requirements.txt` - Python dependencies

## Setup
* Install the latest version of python from python.org
* Create a virtual environment with "python -m venv venv"
* Activate the environment with "venv\Scripts\activate"
* Get the latest version of pip with "python -m pip install --upgrade pip
* Install dependencies from `requirements.txt` with "pip install -r requirements.txt"
* Run the server application with "python server.py"
* Run the client application with "python client.py"
